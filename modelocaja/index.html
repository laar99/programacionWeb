Modelo De Caja

Definicion:

El modelo caja es la manera en que se representan todos los elementos html en una página. Cada elemento genera una caja, el comportamiento de esa caja depende de su clasificación: si es de línea o de bloque.

¿Que es?

es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual.

Propiedades.

Propiedades que controlan el flujo del contenido en una caja.

  • box-decoration-break
  • overflow-x
  • box-sizing
  • overflow-y
  • overflow

Propiedades que controlan el tamaño de una caja

  • height
  • min-height
  • width
  • min-width
  • max-height
  • max-width

Propiedades que controlan los margenes de una caja

  • margin
  • margin-right
  • margin-bottom
  • margin-top
  • margin-left

Propiedades que controlan los rellenos de una caja

  • padding
  • padding-right
  • padding-bottom
  • padding-top
  • padding-left

Otras Propiedades

  • box-shadow
  • visibility
